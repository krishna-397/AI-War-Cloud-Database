# AI War Cloud — project website & database (PRIMARY ARTIFACT)
Author: Sarah Ciston, PhD | 2024–ongoing | Fetched 2026-07-01

- Live project: https://aiwar.cloud/
- About/process/bibliography: https://aiwar.cloud/about.html (full text saved in 02_about-page-process-essay.md)
- Open-source code: https://gitlab.com/sarahciston/aiwar
- Author site: https://sarahciston.com/ | ORCID: 0000-0001-9456-9537

Tagline: "AI decision making from battlefield to desktop."
Self-description: "an interactive database and tangible interactive installation connecting and explaining the current techno-imperial boomerang that is perpetuated by machine learning."

Technical form (relevant to the UI/UX angle): built with Quarto (static publishing from computational notebooks); interactive force-directed network graph (nodes = weapons/companies/ML tasks; adjustable physics simulation, node/edge sizing, export to SVG/PNG/JPG); sortable ITables data table; database of 50+ systems spanning 25 years. Open source, deliberately "readable even in a spreadsheet, to lower the barrier to entry."
